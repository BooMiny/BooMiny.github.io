javascript语言特性

  |  
 阅读次数
  1. javascript语言实现继承机制的核心就是 prototype ，而不是Java语言那样的类式继承。
  2. Javascript 解析引擎在读取一个Object的属性的值时，会沿着 原型链向上寻找，如果最终没有找到，则该属性值为undefined； 如果最终找到该属性的值，则返回结果。
  3. 当javascript解析引擎执行“给一个Object的某个属性赋值”的时候，如果当前Object存在该属性，则改写该属性的值，如果当前的Object本身并不存在该属性，则赋值该属性的值 。
