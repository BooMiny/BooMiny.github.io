PS-批量操作

发表于 2017-06-23 | | 阅读次数 |

一、录制动作

  1. 打开photoshop，选择“窗口”—“动作”，调出动作窗口。

  2. 选择“创建新动作”，点击“记录”，即可在ps中进行操作，ps会自动记录下你的所有操作。

  3. 操作完毕，点击“停止记录”。

二、批量操作

  1. 选择“文件”—“自动”—“批处理”。

  2. 在“播放”选项中，选择录制好的动作。

  3. 在“源”选项中，选择需要操作的图片；在“目标”选项中，选择保存或者保存位置。点击“确定”即可开始批量操作。
